    **Part B: Migrate radius and duration tokens in pages + App.tsx**

    Apply these mechanical replacements in each file:

    **Radius mapping (same pixel values, just semantic names):**
    - `rounded-lg` (8px) -> `rounded-button` (8px) — for buttons, interactive elements
    - `rounded-xl` (12px) -> `rounded-input` (12px) — for inputs, smaller containers
    - `rounded-2xl` (16px) -> `rounded-card` (16px) — for cards, panels, major containers
    - `rounded-3xl` (24px) -> `rounded-card` (16px) — normalizing large containers to card standard (design system specifies 16px for cards)
    - `rounded-full` -> `rounded-pill` (9999px) — for pill shapes, avatars (ONLY where rounded-full is used for pill/badge shapes, NOT for circular elements like w-8 h-8 rounded-full avatars)

    **IMPORTANT for rounded-full:** Do NOT replace `rounded-full` on square elements that need to be circular (avatars with explicit width/height like `w-10 h-10 rounded-full`). Only replace `rounded-full` on non-square elements where it creates a pill shape (e.g., `px-4 py-1 rounded-full` badges/tags). When in doubt, leave `rounded-full` as-is.

    **Duration mapping (same values, semantic names):**
    - `duration-200` -> `duration-fast` (200ms)
    - `duration-300` -> `duration-normal` (300ms)
    - Leave `duration-500`, `duration-700` unchanged (no semantic token exists for these)

    **Per-file instructions:**

    `src/App.tsx` (~5 replacements): Replace rounded-xl/2xl with rounded-input/rounded-card. Replace duration-200/300 with duration-fast/duration-normal.

    `src/pages/Login.tsx` (~10 replacements): Replace rounded-xl/2xl/3xl with rounded-input/rounded-card. Replace duration-200/300. The main card container uses rounded-3xl — change to rounded-card. Keep `md:rounded-[32px]` as a responsive override.

    `src/pages/Dashboard.tsx` (~11 replacements): Replace rounded-2xl/3xl on card containers with rounded-card. Replace duration-200/300 with duration-fast/duration-normal. Leave duration-500/700 animations as-is.

    `src/pages/Leaderboard.tsx` (~8 replacements): Replace rounded-xl/2xl/3xl on panels/tabs/list items. Replace duration-200/300.

    `src/pages/Rewards.tsx` (~8 replacements): Replace rounded-2xl/3xl on reward cards and containers with rounded-card. Replace duration-200/300 with duration-fast/duration-normal. Leave duration-500 hover animations.

    `src/pages/Profile.tsx` (~2 replacements): Replace rounded-2xl with rounded-card.
  </action>
