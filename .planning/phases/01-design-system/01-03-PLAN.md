**A. Color replacements:**

Background:
- `dark:bg-slate-900` -> `dark:bg-surface-dark-primary`

- `dark:bg-slate-700` -> `dark:bg-surface-dark-elevated`

- `dark:bg-gray-700` -> `dark:bg-surface-dark-elevated`
- `bg-white` (card/container bg) -> `bg-surface-primary`
- `bg-gray-50` or `bg-gray-100` -> `bg-surface-secondary`

Text:
- `dark:text-white` -> `dark:text-text-theme-dark-primary`
- `dark:text-slate-100` through `dark:text-slate-200` -> `dark:text-text-theme-dark-primary`
- `dark:text-slate-300` or `dark:text-slate-400` -> `dark:text-text-theme-dark-secondary`
- `dark:text-slate-500` -> `dark:text-text-theme-dark-muted`
- `text-gray-900` -> `text-text-theme-primary`
- `text-gray-600` or `text-gray-500` -> `text-text-theme-secondary`
- `text-gray-400` or `text-gray-300` -> `text-text-theme-muted`

Border:
- `dark:border-slate-700` or `dark:border-slate-600` -> `dark:border-surface-dark-elevated`
- `border-gray-200` or `border-gray-100` -> `border-surface-secondary`

**B. Typography migration (per design-system.md mapping table):**
- `text-2xl font-bold` -> `text-h1`
- `text-xl font-semibold` or `text-xl font-bold` -> `text-h2`
- `text-lg font-semibold` -> `text-h3`
- `text-base font-semibold` -> `text-h4`
- `text-sm` (body text) -> `text-body`
- `text-xs` (captions/hints) -> `text-caption`
- `text-xs font-semibold uppercase` -> `text-label uppercase`

Remove separate font-weight classes when the semantic token already includes fontWeight.

**C. Off-scale spacing:**
Replace any off-scale values (p-2.5, p-5, p-7, p-1.5, etc.) with nearest on-scale equivalents. Leave p-0.5 for micro-alignment.

Rules: No functional changes. Preserve component props, logic, exports. Keep brand-primary accent colors.
