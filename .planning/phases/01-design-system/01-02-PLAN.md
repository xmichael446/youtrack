  <action>
In each file, perform these replacements for dark mode surface colors:

**Background replacements:**
- `dark:bg-slate-900` -> `dark:bg-surface-dark-primary`
- `dark:bg-slate-800` -> `dark:bg-surface-dark-secondary`
- `dark:bg-slate-700` -> `dark:bg-surface-dark-elevated`
- `dark:bg-gray-900` -> `dark:bg-surface-dark-primary`
- `dark:bg-gray-800` -> `dark:bg-surface-dark-secondary`
- `bg-white` (used as page/card bg) -> `bg-surface-primary` (keep, white maps to surface-primary)
- `bg-gray-50` or `bg-neutral-bg` (section backgrounds) -> `bg-surface-secondary`

**Text replacements:**
- `dark:text-white` -> `dark:text-text-theme-dark-primary`
- `dark:text-slate-100` or `dark:text-slate-200` -> `dark:text-text-theme-dark-primary`
- `dark:text-slate-300` or `dark:text-slate-400` -> `dark:text-text-theme-dark-secondary`
- `dark:text-slate-500` -> `dark:text-text-theme-dark-muted`
- `text-gray-900` or `text-neutral-body` (primary text) -> `text-text-theme-primary`
- `text-gray-600` or `text-gray-500` -> `text-text-theme-secondary`
- `text-gray-400` -> `text-text-theme-muted`

**Border replacements:**
- `dark:border-slate-700` -> `dark:border-surface-dark-elevated`
- `dark:border-slate-600` -> `dark:border-surface-dark-elevated`
- `border-gray-200` or `border-gray-100` -> `border-surface-secondary`

**Shadow replacements (where applicable):**
- `shadow-sm` or `shadow` on cards -> `shadow-card dark:shadow-card-dark`
- `shadow-lg` or `shadow-xl` on modals -> `shadow-modal dark:shadow-modal-dark`

**Important rules:**
- Do NOT change any functional logic, state management, or component structure
- Do NOT change any non-color/non-spacing Tailwind classes (flex, grid, rounded, etc.)
- Preserve ALL existing animations and transitions
- If a color is used for a specific non-theme purpose (e.g., brand-primary for accent), leave it as-is
- When uncertain whether a gray shade is "secondary" or "muted", use secondary for content text and muted for hints/captions
