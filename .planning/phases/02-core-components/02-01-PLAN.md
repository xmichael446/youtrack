    **Step 2: Create `src/components/ui/Button.tsx`**

    Props interface:
    ```typescript
    interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
      variant?: 'primary' | 'secondary' | 'ghost' | 'danger';
      size?: 'sm' | 'md' | 'lg';
      loading?: boolean;
      icon?: React.ReactNode;
      iconPosition?: 'left' | 'right';
      fullWidth?: boolean;
    }
    ```

    Variant class maps:
    - **primary**: `bg-gradient-to-r from-brand-primary to-brand-accent text-white shadow-lg shadow-brand-primary/20 hover:shadow-brand-primary/35 hover:scale-[1.01] active:scale-[0.99]`
    - **secondary**: `bg-surface-secondary dark:bg-surface-dark-secondary text-text-theme-primary dark:text-text-theme-dark-primary border border-surface-secondary dark:border-surface-dark-elevated hover:bg-surface-secondary/80 dark:hover:bg-surface-dark-elevated hover:scale-[1.01] active:scale-[0.99]`
    - **ghost**: `bg-transparent text-text-theme-secondary dark:text-text-theme-dark-secondary hover:bg-surface-secondary dark:hover:bg-surface-dark-secondary hover:scale-[1.01] active:scale-[0.99]`
    - **danger**: `bg-status-error text-white shadow-lg shadow-status-error/20 hover:bg-red-600 dark:hover:bg-red-400 hover:scale-[1.01] active:scale-[0.99]`

    Size class maps:
    - **sm**: `px-3 py-1.5 text-caption gap-1.5` (inline actions)
    - **md**: `px-4 py-2.5 text-body gap-2` (default)
    - **lg**: `px-6 py-3.5 text-body gap-2.5` (CTAs, full-width)

    Shared base classes: `inline-flex items-center justify-center font-bold rounded-button transition-all duration-fast disabled:opacity-60 disabled:cursor-not-allowed disabled:scale-100 disabled:shadow-none`

    Loading state: When `loading={true}`, render a `<Loader2 className="w-4 h-4 animate-spin" />` from lucide-react and set `disabled` on the button element. Size the loader icon: sm=`w-3.5 h-3.5`, md=`w-4 h-4`, lg=`w-5 h-5`.

    `fullWidth`: adds `w-full`

    Icon rendering: If `icon` is provided, render it at `iconPosition` (default 'left'). When `loading` is true, replace the icon with the spinner.

    Use `React.forwardRef` so consumers can attach refs. Accept and spread `className` via rest props, appending it after the computed classes.

    Default export: `export default Button`
  </action>
