    Create `src/components/ui/Input.tsx`.

    This component supports both `<input>` and `<textarea>` via a `as` prop.

    Props interface:
    ```typescript
    interface InputProps extends Omit<React.InputHTMLAttributes<HTMLInputElement>, 'size'> {
      as?: 'input' | 'textarea';
      label?: string;
      helperText?: string;
      error?: string;
      icon?: React.ReactNode;
      size?: 'sm' | 'md' | 'lg';
      textareaRows?: number;
    }
    ```

    Size map:
    - **sm**: `px-3 py-2 text-caption` (compact forms)
    - **md**: `px-4 py-3 text-body` (default)
    - **lg**: `px-4 py-4 text-body` (Login-style prominent inputs)

    When `icon` is provided, add `pl-10` (for sm: `pl-9`) to the input and render the icon in an absolutely-positioned container: `absolute inset-y-0 left-0 pl-3 flex items-center pointer-events-none text-text-theme-muted dark:text-text-theme-dark-muted`. On focus-within of the parent group, icon should transition to `text-brand-primary` (use Tailwind `group/input` + `group-focus-within/input:text-brand-primary`).

    Base input classes: `w-full rounded-input border border-surface-secondary dark:border-surface-dark-elevated bg-surface-primary dark:bg-surface-dark-primary text-text-theme-primary dark:text-text-theme-dark-primary placeholder:text-text-theme-muted dark:placeholder:text-text-theme-dark-muted focus:border-brand-primary focus:outline-none focus:ring-4 focus:ring-brand-primary/10 transition-all duration-fast`

    Error state: When `error` is provided, change border to `border-status-error dark:border-status-dark-error` and focus ring to `focus:ring-status-error/10`. Render the error message below the input: `<p className="mt-1.5 text-caption text-status-error dark:text-status-dark-error">{error}</p>`.

    Label: Render above the input: `<label className="block mb-2 text-label text-text-theme-muted dark:text-text-theme-dark-muted uppercase tracking-wider">{label}</label>`.

    Helper text: Render below input (below error if both): `<p className="mt-1.5 text-caption text-text-theme-muted dark:text-text-theme-dark-muted">{helperText}</p>`.

    When `as="textarea"`, render a `<textarea>` element instead of `<input>`, using `textareaRows` (default 4) for the `rows` attribute. Add `min-h-[100px] resize-y` to textarea.

    Use `React.forwardRef`. The outer wrapper is a `<div>` with a `group/input` class. Spread remaining props onto the input/textarea element. Accept `className` for the input element itself.

    Default export.
  </action>
